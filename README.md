# 泰山出品-java版CMS内容管理系统

#### 介绍
泰山出品-java版CMS内容管理系统

#### 软件架构

软件架构说明
| 名称  |  技术栈  |
|---|---|
| 前端模板  |   thyleaf |
| 后端技术  |  springboot  |  
| ORM 框架  |  mybatis-plus  | 
| 缓存技术  |  redis  |
| 安全框架  | Apache Shiro  |
|  日志组件 |   logback |
|  数据库 |  mysql 或者h2（默认） |
| 版本管理  |   git |
| 构建工具  |  maven |
